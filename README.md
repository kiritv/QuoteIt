# Quote It (v4.0.0)

Quote It is a SwiftUI app for discovering and sharing inspirational quotes with optional background photography.

## Highlights
- Random quote on launch
- Swipe left/right to move through quotes
- Browse by author and search quotes/authors
- Favorites and sharing
- Optional background images (Unsplash/Pexels)
- Daily notification
- Quote of the Day on watchOS
